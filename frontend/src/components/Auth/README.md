# Auth Components

This directory will contain the authentication components including:

- BasicRegistrationStep.jsx (with username and fullName fields)
- EmailVerificationStep.jsx
- PersonalInfoStepNew.jsx (with Vietnam address integration)
- UserSurveyStep.jsx
- StepIndicator.jsx
- MultiStepRegister.jsx (main controller)

## Features implemented:
- Separate username and fullName fields
- Vietnam address integration with provinces and wards
- Multi-step registration flow with edit functionality
- Tesla-themed UI design
